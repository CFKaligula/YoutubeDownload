# YoutubeDownload

 To Download Youtube videos and convert to mp3

## Requirements

Download the following packages: moviepy, pytube:

``` powershell
pip install moviepy
pip install pytube3
```

## Usage

To download videos:

``` powershell
./console.py download $URL
```

To download videos as mp3:

``` powershell
./console.py download $URL -mp3
```
